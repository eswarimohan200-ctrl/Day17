# 🚗 AI Vehicle Fuel Cost Dashboard

An AI-powered prompt that converts CSV vehicle fuel data into a responsive HTML dashboard with automated calculations and interactive SVG charts.

## Features

* KPI Cards
* Fuel Cost Analysis
* CO₂ Emission Comparison
* Maintenance Cost Analysis
* Age-Based Trends
* E85 Cost Comparison
* Animated SVG Charts & Gauge
* Responsive Glassmorphism UI

Built for automated data analysis and visualization using HTML, CSS, JavaScript, and AI Prompt Engineering.
